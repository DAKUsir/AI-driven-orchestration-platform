# 🎓 FinishIt — AI-Driven Learning & Career Orchestration Platform

A full-stack AI-powered platform that unifies coding interview preparation, career tools, and learning management into a single cohesive experience. Built with React 19, Node.js, Express, MongoDB, and integrated AI services powered by **GPT-5**.

## ✨ Features

## 🛠 Tech Stack

### Frontend
- **React 19** + Vite
- **Vanilla CSS** — Custom Apple-style dark theme with glassmorphism
- **Framer Motion** — Smooth page transitions & micro-animations
- **Zustand** — Lightweight state management
- **Recharts** — Data visualization (radar, pie, bar charts)
- **Monaco Editor** — Embedded VS Code-like editor
- **Lucide React** — Icon system

### Backend
- **Node.js** + Express
- **MongoDB** + Mongoose ODM
- **Passport.js** — Google OAuth 2.0
- **JWT** + bcrypt — Authentication
- **Multer** — File uploads (resume PDF/DOCX)
- **Socket.IO** — Real-time group chat & voice channels
- **Nodemailer** — Transactional emails

### AI & Integrations
- **GPT-5** via GitHub Models API — Code editor AI assistant, mentor chat
- **Gradio API** (`girishwangikar/ResumeATS`) — Resume analysis, cover letters, interview prep
- **LeetCode GraphQL API** — Problem solving stats sync
- **GeeksForGeeks API** — GFG profile stats sync
- **GitHub REST API** — Repository metadata & categorization
- **YouTube Data API v3** — Playlist & course tracking

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- MongoDB Atlas account (or local MongoDB)
- Google OAuth credentials (for social login)
- GitHub Personal Access Token (for GPT-5 via GitHub Models)

### 1. Clone the repo
```bash
git clone https://github.com/DAKUsir/AI-driven-orchestration-platform.git
cd AI-driven-orchestration-platform
```

### 2. Backend Setup
```bash
cd backend_ai_code
npm install
```

Create a `.env` file in `backend_ai_code/`:
```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
SESSION_SECRET=your_session_secret
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
AI_SERVICE_URL=http://localhost:8000
FRONTEND_URL=http://localhost:5173
BACKEND_URL=http://localhost:5000
GITHUB_TOKEN=your_github_pat_token
ENCRYPTION_KEY=your_64_char_hex_key
EMAIL_SENDER=your_email@gmail.com
EMAIL_PASSWORD=your_app_password
YOUTUBE_API_KEY=your_youtube_api_key
```

Start the backend:
```bash
npm run dev
```

### 3. Frontend Setup
```bash
cd frontend_ai_code
npm install
npm run dev
```

The app will be available at `http://localhost:5173`

## 📄 Resume Analyzer — Gradio API Integration

The Resume Analyzer connects to the `girishwangikar/ResumeATS` Gradio space. All 6 endpoints are exposed:

| Endpoint | Description | Key Parameters |
|---|---|---|
| `/process_resume` | Extract text from PDF/DOCX | `file` (filepath) |
| `/analyze_resume` | ATS score and detailed feedback | `resume_text`, `job_description`, `with_job_description`, `temperature`, `max_tokens` |
| `/rephrase_text` | Rephrase text blocks for impact | `text`, `temperature`, `max_tokens` |
| `/generate_cover_letter` | Create a tailored cover letter | `resume_text`, `job_description`, `temperature`, `max_tokens` |
| `/generate_interview_questions` | Prepare for role-specific interviews | `job_description`, `temperature`, `max_tokens` |
| `/update_job_description_visibility` | Toggle JD requirement | `with_job_description` |
